# legacy

Archived local artifacts and research-only material.

This directory is for content that should be preserved but should not clutter
the core product surface.

Current categories:

- `logs/`: historical run logs.
- `exports/`: old exported packages.
- `qwen_jobs/` and `qwen_offload/`: Colab/teammate offload bundles and scripts.
- `notebooks/`: old notebooks.
- `research/`: literature / research-gap material.
- `direction/`: exploratory research lines and one-off analysis folders.
- `local_ops/`: local machine / thermal / overnight runner helpers.

Do not delete legacy content without explicit approval.
